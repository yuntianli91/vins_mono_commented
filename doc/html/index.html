DBoW2 Library

DBoW2 library for C++: Bag-of-word image database for image retrieval.

Written by Dorian Galvez-Lopez, University of Zaragoza

Check my website to obtain updates: http://doriangalvez.com

Requirements

This library requires the DUtils, DUtilsCV, DVision and OpenCV libraries, as well as the boost::dynamic_bitset class.
